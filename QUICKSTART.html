Method 2: Local Web Server (Recommended)

For full functionality including Google OAuth, run a local web server:

Option A: Python HTTP Server

# Navigate to project folder
cd "Python Test"

# Python 3
python -m http.server 8000

# OR Python 2
python -m SimpleHTTPServer 8000

Then open: http://localhost:8000

Option B: Node.js HTTP Server

# Install (one time)
npm install -g http-server

# Run server
http-server -p 8000

Then open: http://localhost:8000

Option C: VS Code Live Server

  1. Install "Live Server" extension in VS Code
  2. Right-click on index.html
  3. Select "Open with Live Server"

Troubleshooting

Issue: Code doesn't run

Solution: Wait a few seconds for Pyodide to load (first time only). Check internet connection as Pyodide loads from CDN.

Issue: Google Login not working

Solution: Use Demo Login buttons instead. For production, set up Google OAuth credentials.

Issue: Data disappears after closing browser

Solution: Don't use incognito/private mode. LocalStorage is cleared when browser closes in private mode.

Issue: Timer not working

Solution: Check browser console for errors. Ensure JavaScript is enabled.
